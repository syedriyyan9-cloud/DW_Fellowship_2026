<======================> Notes <======================>

What is an array?

An array is a basic data structure that stores a collection of elements, where each item can be identified using at least one index. 
All elements in an array must be homogeneous, meaning they are of the same data type, such as integers or characters. 
These elements are stored in contiguous memory, which means they sit in adjacent blocks next to each other to allow for very fast access. 
In standard C++, arrays typically have a fixed size that is determined when the code is compiled, although dynamic versions also exist.

Memory Representation

In memory, array elements are placed in a sequence of adjacent blocks. 
To find the specific memory address of any element at a given index, a simple formula is used: Address of Array[i] = Base Address + (index * size of data type). 
The Base Address is the starting memory location of the first element (index 0). 
For example, if an integer takes up 4 bytes and the base address is 100, the element at index 1 would be at address 104 because 100 + (1 * 4) equals 104. 
This contiguous layout ensures that if one element is stored at a certain address, the next one is guaranteed to be right after it.

Declaration of Array

To declare an array in C++, you must specify three things: the data type, the name of the array, and its size. 
For instance, writing int numbers; tells the computer to create an array named "numbers" that will hold five integers. 
If you only declare the array without giving it values, it will contain garbage data, which is just random information left over in memory. 
You can also declare arrays for other types, such as characters, by specifying the type and the number of elements it will hold.

Initialization of Array with 4 Methods

There are four common ways to give an array its initial values at the time of declaration. 
The first method is to provide both the size and the specific values inside curly brackets, such as int arr1 = {10, 20, 30, 40, 50};. 
The second method allows you to omit the size in the brackets, and the compiler will automatically count the elements to determine the size for you. 
In the third method, if you provide a size but only list a few values, the remaining spots in the array are automatically filled with zeros. 
The fourth method is a shortcut to set every element in the array to zero by simply writing {0} in the initialization list.

Types of Arrays Based on Size

Arrays can be categorized into two types based on how their size is handled: static and dynamic. 
Static arrays have a fixed size that is set when the program is compiled and cannot be changed later; if you try to add more elements than the size allows, it will cause an error. 
Dynamic arrays, on the other hand, allow the size to be defined or resized while the program is running (at runtime). 
In C++, dynamic arrays are often implemented using pointers or the vector library, which automatically manages the size as you add or remove elements.

Types of Arrays Based on Dimensions

Based on their structure, arrays are usually either one-dimensional or multi-dimensional. 
A one-dimensional array is a simple linear list of elements, like a single row of numbers. 
A two-dimensional array is essentially an "array of arrays" that represents data in rows and columns, similar to a grid or a matrix. 
To declare a two-dimensional array, you provide two sizes, one for the rows and one for the columns, and use nested curly brackets to organize the data. 
While three-dimensional arrays and higher also exist, one-dimensional and two-dimensional arrays are the most commonly used.

Array Traversal

Array traversal refers to the process of iterating through an array to access each element one by one. 
This operation is essential for tasks like printing values, summing them up, or updating the data stored in the array.
In C++, traversal is typically performed using a for loop that starts at index 0 and runs until it reaches the last element at index n−1, where n is the total size of the array. 
During each step of the loop, the index is incremented (i++), allowing the program to move from one adjacent memory block to the next until the entire collection has been visited.

Insertion in Array

To perform an insertion at a specific point in an array, you must shift all subsequent elements to the right to make room for the new value. 
Because arrays are stored in contiguous memory, you cannot simply "squeeze" a value in; 
instead, the process starts from the end of the array, moving each item one position forward until the target index is vacant. 

Once the space is created, the new element is assigned to that index, and the total size of the array is incremented (n++) to reflect the addition of the new data.

Deletion in Array

Deletion is the reverse of insertion and requires shifting all subsequent elements to the left to fill the gap left by the removed item. 
When an element at a specific index is deleted, the program takes the element immediately following it and moves it back one spot, 
continuing this process for all remaining elements until the end of the list. 

After the elements have been shifted, the array size is decremented (n−−), effectively "removing" the last element from the active list even though its value might technically still exist in memory.

Linear Search

Linear search is a fundamental searching technique where you check each element of the array one by one to see if it matches a target value. 
The program uses a loop to iterate through the array, comparing the current element to the target at every step; if a match is found, the search can return the index where the value is located. 
A break statement is often used to stop the loop immediately once the element is found to save time. 
This method has a time complexity of O(n), meaning that in the worst-case scenario, you might have to look at every single element in the array.

Vector Functions

In C++, vectors are dynamic arrays that come with several built-in functions to simplify data management. 
Key functions include: 
push_back, which adds an element to the end of the vector.
pop_back, which removes the last element.
size for counting the current number of elements. 
empty to check if the vector contains any data. 
clear to remove all elements at once. 
insert and erase allow for adding or removing items at specific positions
capacity tells you how much total storage space the vector has currently allocated.
